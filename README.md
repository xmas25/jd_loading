## 如何使用？

依赖library：

 ```dart
dependencies:
  jd_loading: ^0.0.1
  
 ```
在入口页面初始化：

 ```dart
class MyApp extends StatelessWidget {
  double percentage = 0.0;

  @override
  Widget build(BuildContext context) {
    Application.appContext = context;
    //初始化
    JDLoading.init(context);
    return Scaffold(
    );
  }
}
 ```

在任意页面调用，展示loading：

 ```dart
 JDLoading.show();
  
 ```
 更多使用，参见example.
